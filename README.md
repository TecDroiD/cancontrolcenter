BRB Can Control Center
======================
The CCC is a console based ui for sending and receiving can messages on raspberry pi or similar.

needed libraries:
- urwid
- python-can


Installation using virtualenv
-----------------------------
```bash
# mkdir env
# python3 -m venv env
# source ./env/bin/activate
# pip install urwid python-can
```
